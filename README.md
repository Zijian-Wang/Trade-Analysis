# Trade Analysis Tool

A simple web-based calculator to help estimate risk and position sizing for trades.

## Key Features

*   **Position Sizing**: Calculates how many shares to buy based on your risk tolerance.
*   **Risk Metrics**: Shows entry risk percentage and reward-to-risk ratio.
*   **Trade Log**: Keeps a temporary list of your calculated trades for the session.
*   **Export**: Lets you save your trade history as a JSON file.

## How to Use

1.  Open `index.html` in a browser.
2.  Enter your portfolio size and risk preference.
3.  Fill in the ticker, entry price, and stop loss.
4.  See the calculated position size and log the trade if you like.
